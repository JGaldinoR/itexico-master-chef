This is a basic CRUD application made with:

    Front-End - Angular
    Back-End/DB- Firebase (realtime database)

It contains the following funcionalities:

    - Adding new Recipies by clicking on the green button, which will let you add it's informartion such as name, image, desc and ingredients
    - After adding the recipe, this will be saved locally (Meaning it will go away if you refresh the page) but it can permanently be saved with the manage button on the top right corner
    - Manage button will let you Save all the recipes that you have been added locally, update them and also Fetch all the one that you have been saved
    - You can select, edit, delete the recipes locally but you need to save from the manager in order to persist the data
    - The Shopping List works as a to do list basically, it's data only persist locally
    - I added a couple of recipes on the database so you can fetch from the start so have fun!!!
